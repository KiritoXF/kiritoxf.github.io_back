今天的图书馆依旧有很多人

发表于 2017-02-28 | 分类于 杂谈 | 评论：

在图书馆藏的不是书,而是一书架一书架的金砖。

这是一句老话了，很久以前就在书上读到过。但要真正理解这句话的含义，却并不容易。

现在，图书馆早已经成为了自习的地方，学生们大都带着课本、电脑来学习，真正看书的倒是少之又少。不过这似乎也是没办法的事，仔细一想，导致这种现象出现的一个原因，便是教学楼的自习室不允许占座了。大多数人学习都会带很多很多的书，如果来回背着，很沉，基本上都会放在自习的地方，可是自从“颁布”这项政策以来，没处去的同学们就把地点挪到了图书馆， 这倒是无可厚非。

我觉着，课本上的内容不如书架上的书重要。有些课本只是短时间内对我们有用，考过试后便不再重要，甚至可能这辈子都再也不会用到。而书本里的知识却可以留在一个人的气质里，伴随着他的人生，一点点走下去。一个人在离开学校，步入社会后，很难再有那种心情去读书了。对我来说，读科幻的唯一的黄金时期就是大学之前。自从初中邂逅了第一本《科幻世界》，这本小小的杂志伴随了我整个中学时光，我也借由此领略了一个又一个神奇的世界。可是到了大学之后，更多的新鲜事物扑面而来，让人应接不暇。活动、上课、考试、实践，各种各样的事情围绕着你，填满了你的大学时光。好不容易闲下来，唉，看什么科幻呀，打开电脑看看新番、新剧吧。

可要这么讲下去的话，倒与我最初的想法相违了。并不是说上了大学后，我除了上课就是上网了。科幻我还是在读的，王老的《逃出母宇宙》、《天父地母》，何夕的《天年》等，我都读完了。只是那种感觉不一样了。我很难再找到那种浑身的汗毛都要立起来的感觉了。这是一种奇特的阅读体验，只有在中学的夜晚，专注的读一篇科幻小说的我才能感觉的到，这是一种读罢此文，便觉灵魂升华的感觉。但是现在， 我的心不容易静下来了，也不再容易找到这种感觉了。

而走向社会，工作了以后，定会有更多的烦恼扑面而来，让我忙得脚打后脑勺。新的生活环境，新的人际关系，新的日程表，要熟悉这些都够费劲的了，又哪会有时间读书？

这就是我一直强调的，在大学应该多读书，多读好书，多读经典。当一个人感到迷茫，不知道自己该做什么的时候，不用多想，去读书。这是一件永远不会错的事情。如果说，你去找个兼职，赚点零花钱，我就可以反驳要先填满自己的脑袋，再去填满自己的口袋，这是东师大曹胜高教授的一句话。曹老师的意思自然不是要你把脑袋填得满满的，再也装不进去东西了才罢休。而是说，毕业了以后有很多时间去赚钱，但是读书、学习的时间就不多了。我认为，除非了家里有特殊情况，比如父母负担很重，自己打工可以减轻家庭负担，等等。如果是这样，那自然要去，因为这是传统美德，是孝道。可若是说，嗯，还凑合过得去，还有半年就要参加工作了，等等。若是这样，可就断了那个念头吧！在学校你有更重要的事情可以做。这也是我看尔雅以来，学到的最重要的东西之一。

如果说，你在宿舍打打游戏，混过一天是一天，那都不用等我反驳了，有很多的人要反驳你。老师、父母，还有你的内心。依我看，就是去外面跑两圈，都比窝在宿舍打电脑强。跑步还能锻炼身体，磨练意志呢！而打游戏，只会给你带来更多的疲惫感，摧残一个人的意志。除非说，你的技术很好，能以此维生，一场比赛随随便便就几十万、甚至几百万什么的，不然就不要整天沉迷在那里，没好处。偶尔碰碰，调节调节心情，劳逸结合，也就罢了，不能全指着那个不是。

可如果说，你拿起书本，找几篇经典的文章读上一番，去图书馆找几本经典名著读个十遍八遍，那我即使有天大的本事，也找不出什么话可以反驳了。大量的阅读，也许书中的内容不久就忘记了，但是书中的某些东西，比如说作者的灵魂，主人公的精神，会留在你的脑海里，多年之后会成为你的一部分，你的举手投足，言谈举止都会体现出来，你所读过的书。

我也经常劝说身边的朋友，多读读书。也常跟同学说，嘿！我今天读了一本书，可有意思了，有这么这么这么个情节。我以前有个愿望，就是能“消灭”图书馆一书架的书，可现在看来，却不是轻易能实现的了。毕竟文学方面的书不同于技术类的书，前者马虎不得，而后者看多了就能快速的浏览，跳着看了。
