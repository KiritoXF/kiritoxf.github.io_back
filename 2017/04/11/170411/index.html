我不喜欢大风天

发表于 2017-04-11 | 分类于 杂谈 | 评论：

温暖的阳光，和煦的春风，慢步到图书馆。

这是前天我的感受。中午吃完饭后，阳光正好，微风拂面，手托着衣服就这样慢慢的走到图书馆，别提有多舒服了。

但是，今天就变了一个样子。早上出门时还一点风都没有，到了图书馆后，就看到外面的树在疯狂的抖动，空气也颤抖着。去吃饭时，在路上走着，被如此之大的风吹着，人是会失去希望，失去斗争下去的勇气的。

我们学院有一座桥，名叫学苑桥，是连接生活区和教学区唯一的通路。这座桥上铺的是铁皮，而今天这是第二次被吹跑了好几块。我越来越担心走在桥上的我们的安全了。

昨天打开 GitHub 时，忽然发现自己已经快两周没有 commit 了，也就是两周都没更新博客了。因为我全部的 commit 就是部署 hexo 了。在这段时间里，我前半学期的两门课已经走到了尾声，下周和大下周就要考试了。一门是网络综合实验，一门是 UML，老实说这七、八周以来，我没感觉记住了多少东西。实验课由于有实际操作，即使不知道自己在干什么，也能有点印象，比如：啊！原来线插到这个口就有网了。而 UML，其实还是软件工程的那些内容，老师上课也一直在讲故事，真正课上的内容可能只需要一半的时间就能全部讲完。不过，跟上学期数据库老师不同，虽然都是在“讲故事”，“扯别的”，可蒋老师讲得都挺有意思的。蒋老师经常讲他公司的事情，听着也能了解一些。

在这段时间里，我读完了《大师和玛格丽特》，一本在整个大工的图书馆都找不到的书。这是一本有关讽刺、批判的小说，作者布尔加科夫在当时的文学界不受好评，受排挤，感到无处可去。这本书也是在他死后几十年后才得以面世。我是因为前段时间在读余华和王小波，也忘记是哪位在他的作品中推荐了这本书，才找来读了。

这两周，我做了三家公司的笔试，完美世界、今日头条和京东。除了京东我投的是C++，比较正常外，完美和今日头条我投的分别是文案策划和数据分析师。而笔试内容就很不正常了，一个考我古诗词，一个考我数据分析、建模什么的。这就告诉了我，职位要慎重投递。那么直到现在，包括一月份投的埃森哲，都没有任何消息。倒是有别的几家拒绝了，有的是人招满了，有的是HR不感兴趣。下周 LINE 要来软院宣讲，现场笔试，不知道会是个怎样的结果。

我还是在坚持来图书馆的。其实现在这么回头一想，也没发现自己做了什么有意义的事。无非是了解了一下前端，HTML、JavaScript 之类的，知道了 bootstrap 和 AmazeUI 这两个前端框架，试着用模版搭了几个页面出来。然后把 Java 的视频看完了，还胡乱学了些其它的东西。上午基本上是在学日语的，在虎将题库上刷题，词汇的正确率只有可怜的 20%，语法和听力还稍稍好一点，阅读也比较一般，我也越来越看不清未来的走向了。

在玩过炉石传说、Fariea、Spellware、卡片怪兽、Dyllest** 后，我发现还是万智牌最好玩。可惜的是身边竟然一个一起玩的人也找不到。这周六有万智牌体验日的活动，我问了很多人，也都没有愿意去的。

我是不喜欢大风天的。虽然我给摘录随笔本起名叫“风语集”，可我是不喜欢大风的。在开头我也说，和煦的春风吹在身上让人感到舒服，而令人不安的春风打在脸上就一点也不舒服了。如果我穿上冲锋衣（这是一种防风效果很好的衣服），我就不会说这种话了，只要拉好拉链，戴上帽子，刮再大的风都跟我没有关系。
