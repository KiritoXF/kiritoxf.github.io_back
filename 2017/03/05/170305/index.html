我为什么喜欢一个人

发表于 2017-03-05 | 分类于 杂谈 | 评论：

永远不要相信室友说要七点半起床跟你一起去建设社会主义的鬼话

照理来说，大多数的时候，我都是喜欢独来独往的。也许跟性格有一定的关系，但主要是觉得这样最省事、方便。一个人的话，去哪里，做什么，什么时候去，都是自己说的算的。可要是两个人，就要顾虑到对方的想法，做一些取舍，不过有时候对方还是会依着自己的。可要是三个人往上，那事情就不一样了。三个人开始，就变成了一个小集体，做决定时就不是简单的加减法了，会变得很麻烦，除非这三个人中有一个人怎么样都可以，全依着另外两个人，那这就变成了两个人的问题。可多数时候并不是这样，所以会产生很多不必要的纠结，这就是我不喜欢跟别人在一起的原因。

昨天晚上，我跟室友讨论明天要去做什么，因为周末总不能在窝在寝室一整天。后来决定要去自习室，因为图书馆周末有一个神奇的开门时间：上午十点。由于本篇意不在此，便不再多讲。既然图书馆这个开门的时间很“尴尬”，我们也就只好去教学区的自习室了，好在B区是有插排的，可以用电脑。结果这厮今早竟没起来，我看他没起来心想，再等等，反正是周末。就这样等着等着的时间就过去了一个小时，后来我一看，这也不是办法，就只好自己起来了。结果九点才来到教学区，开始敲这篇文章。

今天早饭，已经是差一刻钟就九点了，食堂已经没有什么太多的选择了，我在电视前的桌子边坐下，前面桌应该是一对情侣。我这个人吃饭是很慢的，但小时候却不是这样。小时候上学着急，早饭通常十几分钟就搞定，午饭时间较短，也得吃得快点。上大学后，时间就不是太紧了，也就渐渐的越吃越慢，昨天跟同学去外面吃，旁边的同学都吃完两碗饭了，我这一碗都还没吃完。不过这也不能怪我，汤里骨头太多。所以说，今天的早饭我也是吃的不快的，但直到我吃完，前面那一桌都还在吃。这时间就花的挺多了，大早上这么好的时间。这要是一个人就不会这样。一个人的话，吃多快、吃多慢，都跟别人没关系，都只是自己的时间。要是跟别人一起吃的话，你先吃完了等对方，对方吃完了等你，两个人再唠上两句，这饭一时半会儿就吃不完。就好像过年跟亲戚聚一起吃饭，一桌子菜要吃上一个下午之久，难以想象。这个例子似乎不太恰当，毕竟两者目的不同，前者是为了增进陌生人间的感情，后者是为了增进亲友间的感情。但其实一想，也许是我的想法不对呢？人家俩人在一块吃早饭，肯定不是只是在吃早饭，就像双关语似的，说出来的意思跟实际的意思不一样。没有相关经历的我可能不会理解，因为我也只是站在自己的角度去看这个问题罢了。

昨天晚上决定去哪吃饭的时候，我是这样想的。食堂肯定要被pass掉，因为如果不是没有别的选择的话，我是不想再去食堂了。我已经一个假期没有吃大连大学外面的那家重庆小面了，因此昨天格外想吃。不只是昨天，今天也想吃，因为昨天并没有吃到。于是就跟游桑商量，咱们运动完去大大那吃点啥呀？游桑说，好啊，吃什么呢。我说，重庆小面怎么样，不过香锅也可以。这说明我那个时候不只是想吃面，香锅也很想吃。但后来梁桑不愿意吃重庆小面，而想到中冶北方那看看有没有什么可以吃的。我一想，重庆小面我自己什么时候都可以去，但是中冶北方那自己可就去不了了，因为我以前没有去过，所以便不知道要去哪里，有就同意了。后来在那一下车，嗬，好多的人，好多的女生。在那走了走，我是没看到什么想吃的，梁桑看一家排骨米饭还不错，我一想，起码有汤呐，天怪冷的也不错，也就进去了。于是便出现了上文的那一幕，我点的黄豆排骨汤里，果然都是排骨，只不过排骨只有后边的“骨”字，肉却不知道哪里去了。这一顿饭也算创造了一个难得的奇迹：我从来不知道自己可以吃这么多的饭。

由此看来，人多了也有人多了的好处，虽然这顿饭让我吃出来很多骨头，但是汤确实是真的，十二块的价钱也还挺划算。此行也让我发现了一些新的地方，比如说，中冶北方那原来也有家乐哈哈超市，还有这家排骨米饭。其实他家如果不点黄豆汤的话，其它的东西应该还挺实惠，只不过下次我要是再去那吃饭，就要自备餐具了——那家店神奇的清一色一次性筷勺。

写到这里的话，这篇文章的题目可就要改成“一个人好还是许多人好”了。可究竟为什么这么写下来了，我也没想清楚。
