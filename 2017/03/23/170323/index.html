Git 使用备忘录

发表于 2017-03-23 | 分类于 博文 | 评论：

首次将本地代码提交至 GitHub，所需要的命令等。

  1. 到代码所在文件夹，git init，创建 git 仓库。
  2. 使用 git add * 将文件添加到本地仓库中。（类似加入到缓存区中）
  3. 使用 git commit -m "test" 将文件提交到本地仓库。（类似正式提交进本地仓库中）
  4. 使用 git remote add origin url 将本地仓库关联到 GitHub 上，url 为你仓库的 https 链接
  5. 首次需要用 git pull --rebase origin master
  6. 然后再 git push -u origin master 即可提交完成

以后再将代码更新，所需要的命令。

  1. git add *
  2. git commit -m "test"
  3. git push origin master

以上内容参考 教你上传本地代码到github
